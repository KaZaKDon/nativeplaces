# Родные Места — статус проекта

## Что уже сделано

### Базовая архитектура

Проект построен на React + Vite.

Основные страницы:

- `HomePage` — главная cinematic-страница проекта.
- `CategoriesPage` — страница категорий мест.
- `MapPage` — карта с объектами, фильтрами и sidebar.
- `PlacePage` — отдельная страница объекта.
- `SubmitPage` — форма добавления места.
- `SubmitLocationPage` — выбор точки на карте.
- `AccountPage` — заготовка личного кабинета.

---

## Карта

Реализовано:

- Leaflet + OpenStreetMap.
- Маркеры объектов.
- Цвета маркеров по категориям.
- Hover marker ↔ sidebar sync.
- Active marker system.
- Автофокус карты по фильтру, поиску и выбранному объекту.
- Sidebar справа на desktop.
- Bottom sheet на mobile.
- Фильтрация по категории через URL.
- Поиск по названию, описанию, категории, тегам.
- Debounce поиска.
- Нормализация поиска `ё/е`.

---

## Категории

Реализовано:

- Страница категорий.
- Фон folded map.
- Карточки категорий с фоновыми изображениями.
- Переход из категории на карту с фильтром.
- Кнопка открытия всей карты.
- Планируется кнопка `Добавить место`.

---

## Страница объекта

Реализовано:

- Маршрут `/place/:slug`.
- Hero-блок с изображением.
- Hero-слайдер фотографий.
- Fullscreen viewer фотографий.
- Стрелки вперед/назад.
- Счетчик фотографий.
- Кнопка `Открыть на карте`.
- Кнопка `Построить маршрут`.
- Кнопка `Сообщение автору`.
- Модалка сообщения автору в виде простого диалога.
- Теги объекта.
- Цена и дополнительные параметры.
- Возврат к карте с сохранением категории и выбранного объекта.

---

## Добавление места

Реализовано:

- Страница `/submit`.
- Выбор основной категории.
- Выбор типа объекта внутри категории.
- Динамические поля по категории.
- Базовые поля:
  - название;
  - краткое описание;
  - полное описание;
  - адрес или ориентир;
  - контакт;
  - фотографии.
- Выбор координат через отдельную страницу `/submit/location`.
- Сохранение координат в `localStorage`.
- Сохранение добавленного объекта в `localStorage`.
- Добавленные объекты появляются на карте в текущем браузере.

---

## Личный кабинет

Реализовано:

- Страница `/account`.
- Карточки:
  - Мои места;
  - Сообщения;
  - Избранное;
  - Профиль.
- Пока без авторизации и backend.
- Кнопка `Кабинет` добавлена на главную.
- Кнопка `Войти` пока скрыта через CSS.

---

## Что пока работает в demo-режиме

- Новые места сохраняются только в `localStorage`.
- Фото через форму пока не сохраняются.
- Сообщения автору пока не отправляются.
- Личный кабинет пока статический.
- Авторизация пока не реализована.
- Модерация пока не реализована.
