## Личный кабинет

Работает:

### Мои места

* список объявлений
* создание
* редактирование
* удаление
* архив

### Избранное

Работает через БД.

Показывает количество избранных объявлений.

### Сообщения

Реализовано:

* создание диалога по объявлению
* первое сообщение уходит владельцу объявления
* ответы владельца
* просмотр истории сообщений
* автообновление открытого диалога каждые несколько минут
* количество диалогов отображается в кабинете

Исправлено:

* ошибка отправки самому себе
* ошибка SQL параметров
* ошибки useState/useEffect
* ошибки отображения диалога

Остаётся:

* непрочитанные сообщения (не реализовано)

### Обращения

Полностью переведены с localStorage на БД.

Типы:

* Поддержка
* Предложение

Реализовано:

* создание обращения
* просмотр своих обращений
* просмотр статуса
* просмотр ответа администрации

Статусы:

* new
* in_work
* closed

### Жалобы

Работают.

Пользователь может отправить жалобу на объявление.

### Отзывы

Переведены на постмодерацию.

Текущая схема:

* отзыв публикуется сразу
* модератор может скрыть/удалить позже
