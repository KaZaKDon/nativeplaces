# Native Places — статус на 05.06.2026

## Что сделано по маршрутам

### Backend

Готовы API:

```txt
api/routes/index.php

api/routes/show.php
api/routes/add-place.php
api/routes/remove-place.php
api/routes/update.php
api/routes/delete.php
api/routes/reorder.php
api/routes/share.php
```

### Возможности

#### Маршруты

* Создание маршрута
* Редактирование маршрута
* Удаление маршрута
* Получение списка маршрутов пользователя
* Получение одного маршрута
* Публичный / личный маршрут
* Share token для каждого маршрута

#### Точки маршрута

* Добавление объекта в маршрут
* Удаление объекта из маршрута
* Сортировка точек (backend готов)
* Получение списка объектов маршрута

#### Публичные маршруты

Работает:

```txt
/api/routes/share.php?token=...
```

Возвращает:

* маршрут
* список объектов маршрута

Только для:

```txt
is_public = 1
```

### Страница маршрута

Файл:

```txt

```

* описание
* список объектов
* адрес объекта
* категорию объекта
* изображение объекта

Кнопки:

```txt
Открыть
Удалить
Построить маршрут
Поделиться маршрутом
```

---

### Добавление объекта в маршрут

Из карточки объекта:

```txt
Добавить в маршрут
```

Открывается модальное окно:

```txt
AddToRouteModal
```

Пользователь выбирает маршрут.

Используется:

```txt
POST /api/routes/add-place.php
```

Работает.

---

### Построение маршрута

Используется:

```txt
openYandexRouteFromCurrentLocation()
```

Логика:

Если геолокация разрешена:

```txt
Моё местоположение
↓
Точка 1
↓
Точка 2
↓
Точка N
```

Если геолокация запрещена:

```txt
Точка 1
↓
Точка 2
↓
Точка N
```

---

### Поделиться маршрутом

Добавлена кнопка:

```txt
Поделиться маршрутом
```

Копирует в буфер:

```txt
https://native-places.ru/routes/share/{token}
```

Используется:

```js
navigator.clipboard.writeText(...)
```

Работает.

---

### Публичная страница маршрута

Создано:

```txt
SharedRoutePage.jsx
```

Маршрут:

```txt
/routes/share/:token
```

Локально работает.

На хостинге необходимо проверить после очередной сборки:

```bash
npm run build
```

и загрузки нового dist.
