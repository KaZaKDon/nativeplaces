# Ростовская область — стартовый справочник населённых пунктов

Дата: 2026-06-23

Цель файла — быстро заполнить таблицу `localities` для запуска пользовательского добавления объявлений. Это не полный официальный справочник всех населённых пунктов Ростовской области, а практичный стартовый набор: города, рабочие посёлки, районные центры, крупные станицы/сёла/посёлки и несколько крупных хуторов.

## Источники для сверки

* ОКТМО Росстата / классификатор ОКТМО — основной источник для последующего полного импорта.
* Публичные списки населённых пунктов Ростовской области: используются только для быстрой первичной сверки названий и крупных населённых пунктов.

## Важно перед импортом

1. Если таблица `localities` уже содержит часть записей, импортируйте через `INSERT IGNORE` или `ON DUPLICATE KEY UPDATE` по уникальному `slug`.
2. Координаты можно добавить позже: в текущей модели они nullable.
3. Для быстрого запуска достаточно `region`, `district`, `title`, `slug`, `sort_order`.
4. Полный справочник лучше позже импортировать из ОКТМО/ФИАС, а этот файл использовать как стартовую ручную seed-выборку.

## Что добавить позже

Для полноценного справочника лучше сделать отдельный импорт из ОКТМО/ФИАС и добавить:

* официальный код ОКТМО;
* тип населённого пункта (`город`, `станица`, `село`, `хутор`, `рабочий посёлок`);
* координаты центра;
* `is_active` и `sort_order`;
* альтернативные варианты написания для поиска.
